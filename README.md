## Controles
- **Setas ou W/D**: Mover peça para esquerda/direita
- **Seta para baixo ou S**: Acelerar descida
- **Setas para esquerda e direita ou Q/E**: Rotacionar peça para esquerda/direita
- **Espaço**: Enviar peça para o fundo
- **ESC**: Sair do jogo
- **R**: Ver ranking (no menu inicial)

### Pré-requisitos
- Python 3.7 ou superior
- Pygame
